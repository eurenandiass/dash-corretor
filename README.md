# Nexus Pro | CRM & Simulador de Inteligência Imobiliária

O **Nexus Pro** é uma Single Page Application (SPA) desenvolvida para demonstrar competências avançadas em lógica de programação, manipulação de dados e experiência do usuário (UX).

## 🚀 Tecnologias & Bibliotecas
- **Vanilla JavaScript:** Lógica de negócio pura, sem dependências de frameworks pesados.
- **Tailwind CSS:** Interface moderna em modo Dark/Gold.
- **Chart.js:** Visualização de dados dinâmica e gráficos de performance.
- **SweetAlert2:** Sistema de notificações e feedbacks premium.
- **LocalStorage:** Persistência de dados local para gestão de leads.

## ⚙️ Funcionalidades de Elite
- **Autenticação de Segurança:** Tela de login funcional com validação de credenciais.
- **CRM Inteligente:** Sistema de gestão de leads com "Termômetro" de interesse editável em tempo real.
- **Simulador Financeiro Bancário:** Motor de cálculo que suporta sistemas **SAC** e **PRICE**, utilizando fórmulas reais de juros compostos e equivalência de taxas.
- **Data Viz:** Gráficos interativos que mostram a proporção de juros e capital no financiamento.

## 🛠️ Instalação e Teste
1. Acesse o link do projeto.
2. Utilize as credenciais de teste: `admin` / `admin`.
3. Navegue pelos menus para testar a reatividade da interface.
